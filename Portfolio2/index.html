My Services

Web Developer

My main job as a web developer is to create and maintain websites using programming languages such as HTML, CSS, JavaScript and various frameworks and libraries, as well as working in teams and developing websites, interacting with clients, documenting the code created and providing maintenance.

Web Designer

My main task as a web designer is to create aesthetically pleasing websites and applications that are easy to use and understand. I work with graphics programmes such as Photoshop, Illustrator and InDesign, create web page layouts, define style, colour schemes and typography, and design icons and graphical interface elements. I take into account user experience and design interfaces that ensure usability and user satisfaction.

UI/UX Designer

My main task as a UI/UX designer is to develop user-friendly and intuitive interfaces for websites and applications. I am able to analyse user and market needs, conduct research, create user interface prototypes and layouts, and design application logic and navigation. I take into account visual design, colour schemes, typography and layout of interface elements.

Artist/Digital Artist

I have a good sense of style and aesthetics in order to create graphic images that meet the given requirements and project objectives.
